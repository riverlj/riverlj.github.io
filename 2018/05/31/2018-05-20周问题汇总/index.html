libuv移植的iOS平台

libuv是一个跨平台的，高性能的，事件驱动的I/O库。具体教程见libuv。

具体步骤：

  1. 安装Python
  2. 下载libuv源码
  3. 在终端中进入libuv目录。
  4. 创建build目录。mkdir -p build
  5. 下载gyp https://chromium.googlesource.com/experimental/external/gyp
  6. 执行 ./gyp_uv.py -f xcode，会在在output目录生成uv.xcodeproj
